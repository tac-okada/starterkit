.util templateCSS

update：230815

応募フォームや応募要項など、キャンペーンサイト等でよく使うパーツを中心にユーティリティ化しました。基本的にCSSを触らずに実装・修正できることを目指してします。
クライアントサイトの一部に適用するケースを考慮し、クラス.utilで囲った範囲のみに適用されます。

  • 制作物のクオリティアップ
    • 時間をかけるべきところに時間を使えるようにしたい
  • コーディングのスピードアップ
    • 極力CSSを触らずにコーディングできること
    • 定期的にメンテナンスし検証不要な状態を保つ
  • 誰でも理解できること
    • 当ページにサンプルを網羅します

使い方

CSSの冒頭で以下4つを読み込み、サイトに応じて拡張して使うことを想定しています。

@use "setting";
@use "reset";
@use "common";
@use "util";

.util{
〜〜 ここで拡張します（サイトに応じたデザイン調整など） 〜〜
}

html内の任意の箇所を.utilで囲います。

<div class="util">
〜〜 ここに適用されます 〜〜
</div>

クラス名について

  • 通常のクラス名（ハイフンから始まらないもの）
    • 基本的にどこにでも適用可能(utility)です
  • ハイフンから始まるクラス名
    • 特定のクラスに依存するスタイル(modifier)です

レスポンシブ対応

  • 768px以上（PC・タブレット）
    • pxでサイズ固定されます
  • 767px以下（スマホ）
    • vwで画面サイズに応じてリサイズされます（PC時のpx値の関係性は維持します）

フォントサイズ

クラスfsize○○で指定します。
デフォルトは16pxですが、10px〜22pxまで1px単位で指定可能です。
見出しを除き、各要素にフォントサイズは定義していないので、全てこのクラスを付与して個別に指定します。
またline-heightはクラスlh○○で指定します。デフォルトは1.8ですが、1〜2.2まで0.2刻みでlh10〜lh22の記述で個別に指定できます。

<p class="fsize16 lh18">テキスト</p>

↓↓↓↓

font-size: 1rem !important;
line-height: 1.8 !important;

余白

余白（padding/margin）は縦のみ定義しています。クラスmt○○mb○○pt○○pb○○で指定します。0〜100pxまで5px単位で指定可能です。
基本的に各要素に縦の余白は定義していないので、全てこのクラスを付与して個別に指定します。
またmargin-topのみマイナス値も指定可能です（-0〜-100px）。

<div class="mt-10 mb0 pt20 pb15"></div>

↓↓↓↓

margin-top: -10px !important;
margin-bottom: 0px !important;
padding-top: 20px !important;
padding-bottom: 15px !important;

buttonタグの場合

buttonタグの場合はタグを内包できるので、使い勝手を考慮して一般的な応募フォームなどではbuttonタグの使用をおすすめします。

<button type="submit" class="btn mt20"><span class="fsize20 bold">buttonで実装したボタン</span></button>

inputタグの場合（非推奨）

inputタグを使用する場合タグを内包できないので、必ずボタン自体にフォントサイズを指定します。改行もできません。

<input class="btn fsize20 mt20" type="submit" value="inputで実装したボタン">

横並び

親要素に「.btnArea」を指定することでPC時に横並びになります。SP時は左のボタンが下、右のボタンが上になります。

戻る 進む 
<div class="btnArea mt20">
  <a class="btn -sizeS -nega" href=""><span class="fsize20 bold">戻る</span></a>
  <a class="btn" href=""><span class="fsize20 bold">進む</span></a>
</div>
